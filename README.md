# Docker-Init
A simple bash script to initialize an Apache Server and MySQL Server with phpmyadmin for your **localhost project**.

## Built With
* Bash

## Requirements
* Linux OS ( Tested on [Ubuntu 20.04](https://ubuntu.com/) )
* [Docker](https://www.docker.com/)

## Usage
1. Drop the script on your project root.
2. Run the next command on terminal in project root.

## Command
```
$ sudo bash ./docker-init.sh

```
